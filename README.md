# EU5QOLCollection
Quality of life improvements utilities collection mod of eu5 named ADAC. The ADAC collection provides a series of quality-of-life improvements. Currently, it includes tools for removing useless buildings and updating outdated buildings. All features are designed purely for convenience and do not affect game balance. Unfortunately, Ironman compatibility is not supported. None of these operations will reduce your treasury below 10,000.

ADAC 系列提供了一组用于提升游戏体验的便捷功能。目前包括无用建筑移除与过时建筑升级等操作。所有功能仅用于改善操作体验，不会对游戏平衡产生任何影响。由于机制限制，本模组不支持铁人模式。所有操作均不会使你的国库资金低于 10,000。
